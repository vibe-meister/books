Navier–Stokes 3D Existence & Smoothness – Blow-up Strategy Log
===============================================================

THE DEARMAN CASCADE: BLOW-UP METHOD

This plain-text file mirrors the markdown log, recording the exploration tree, thought process, and key equations for a finite-time blow-up construction attempt focused on axisymmetric flow with swirl and non-Type-I similarity scenarios. This construction is called the DEARMAN CASCADE: blow-up method, named for its novel multi-point, wave-mediated cascade mechanism.

1. Core Equations
-----------------

3D incompressible Navier–Stokes (viscosity scaled to 1):

  ∂_t u + (u · ∇)u = -∇p + Δu
  ∇ · u = 0

on R^3 or T^3, with smooth, divergence-free initial data u_0.

Energy identity (formal, no forcing):

  (1/2) d/dt ∫ |u|^2 dx + ∫ |∇u|^2 dx = 0

Clay problem: either prove global smoothness for all such u_0 or construct a counterexample with finite-time blow-up.

2. Top-Level Choice: Regularity vs Blow-up
------------------------------------------

Two logical directions:

  A) Global regularity: a priori bounds prevent blow-up.
  B) Blow-up: explicitly construct smooth initial data whose solution becomes singular in finite time.

We chose B) Blow-up.

3. Blow-up Strategy Options
---------------------------

We considered four main strategies:

  B1) Self-similar or discretely self-similar blow-up.
  B2) Symmetry-driven blow-up (axisymmetric with swirl, helical, etc.).
  B3) Convex-integration wild solutions (Euler/NSE variants).
  B4) Rigorous numerics in a special ansatz.

We chose B2) Axisymmetric with swirl.

4. Axisymmetric Navier–Stokes with Swirl
----------------------------------------

Use cylindrical coordinates (r, θ, z) and write velocity:

  u(r, θ, z, t) = u_r(r, z, t) e_r + u_θ(r, z, t) e_θ + u_z(r, z, t) e_z

Assume axisymmetry:

  ∂_θ(·) = 0.

Incompressibility:

  (1/r) ∂_r (r u_r) + ∂_z u_z = 0.

Momentum equations (schematic, unforced):

Radial:
  ∂_t u_r + u_r ∂_r u_r + u_z ∂_z u_r - (u_θ^2 / r)
    = -∂_r p + (Δ u_r - u_r / r^2)

Swirl:
  ∂_t u_θ + u_r ∂_r u_θ + u_z ∂_z u_θ + (u_r / r) u_θ
    = (Δ u_θ - u_θ / r^2)

Axial:
  ∂_t u_z + u_r ∂_r u_z + u_z ∂_z u_z
    = -∂_z p + Δ u_z

with the axisymmetric Laplacian

  Δ = ∂_r^2 + (1/r) ∂_r + ∂_z^2

on scalar components.

Swirl intensity:

  Γ(r, z, t) = r u_θ(r, z, t)

obeys a transport-diffusion-reaction type equation, and is a natural quantity to track for potential blow-up.

Known facts:
  - Axisymmetric flow without swirl (u_θ ≡ 0) is globally regular.
  - Axisymmetric with swirl is an open problem; any blow-up must leverage swirl, likely near r = 0.

5. Self-Similar Blow-up Route (B2.1)
------------------------------------

We explored self-similar / discrete self-similar mechanisms within axisymmetric swirl.

Standard backward Type-I self-similarity (blow-up at t = T, x0 = 0):

  u(x, t) = (T - t)^(-1/2) U( x / sqrt(T - t) )
  p(x, t) = (T - t)^(-1)   P( x / sqrt(T - t) )

Let y = x / sqrt(T - t). Then U(y) solves:

  - (1/2) y · ∇U - (1/2) U + (U · ∇) U = -∇P + ΔU
  ∇ · U = 0

In axisymmetric form, U has components (U_r, U_θ, U_z) independent of θ.

Finite-energy constraint:
  - To have u_0 ∈ L^2, U must decay sufficiently fast at infinity (faster than |y|^(-1)).

Liouville/non-existence results (e.g., Nečas–Růžička–Šverák) show:
  - Under certain critical integrability assumptions (e.g., U ∈ L^3 or certain Morrey spaces), the only backward self-similar solution is U ≡ 0.

Conclusion:
  - For Clay-admissible data (smooth u_0 ∈ L^2), classic Type-I self-similar blow-up is strongly disfavored and largely ruled out.

6. Fork: S2 (Off-Clay) vs S1 (Clay-Admissible)
----------------------------------------------

Given the above:

  S2) Relax integrability; allow infinite-energy initial data to build a self-similar singular U.
      - Could yield interesting singular solutions, but not a Clay example.

  S1) Maintain Clay conditions (smooth, finite energy).
      - Abandon strict Type-I self-similarity.
      - Consider discrete self-similarity (DSS) or Type-II blow-up (non-standard scaling).

We chose S1.

7. Rescaled NSE and Discrete Self-Similarity (S1-DSS)
-----------------------------------------------------

Introduce similarity variables for potential blow-up at (0, T):

  y   = x / sqrt(T - t)
  τ   = -log(T - t)
  v(y, τ) = sqrt(T - t) u(x, t)
  π(y, τ) = (T - t) p(x, t)

Then u(x, t) = (T - t)^(-1/2) v(y, τ).

Plug into NSE to get rescaled NSE:

  ∂_τ v = Δv - (1/2) y · ∇v - (1/2) v - (v · ∇) v - ∇π
  ∇ · v = 0

A Type-I self-similar blow-up corresponds to a steady state:

  ∂_τ v ≡ 0 ⇒ v(y, τ) = V(y).

Instead, we seek more complex behavior as τ → ∞.

Discrete self-similarity (DSS):

Original DSS relation for u(x, t) with factor λ > 1:

  u(x, t) = λ u( λ x, T + λ^2 (t - T) )

In rescaled variables, this translates to:

  v(y, τ + T_λ) = v(y, τ)

where T_λ = 2 log λ.

So a DSS blow-up corresponds to a nontrivial τ-periodic solution of rescaled NSE.

8. Dynamical Systems View
-------------------------

Let S(σ) be the semigroup solving rescaled NSE:

  v(·, τ) = S(τ - τ0) v(·, τ0).

Discrete self-similarity with factor λ corresponds to a periodic orbit of period T_λ:

  v*(·, τ + T_λ) = v*(·, τ).

In terms of the Poincaré map P = S(T_λ):

  v* = P(v*), v* ≠ 0.

The analysis challenge is to:
  - Choose a function space X (axisymmetric, divergence-free, with suitable decay) that’s invariant under the rescaled flow.
  - Study the operator P: X → X, looking for nontrivial fixed points using spectral analysis, bifurcation theory, or topological degree.

Obstacles:
  - Ensuring that P is compact or has suitable continuity properties.
  - Avoiding existing Liouville/non-existence results that might rule out bounded ancient solutions in certain norms.

9. Type-II Blow-up Sketch (S1-T2)
---------------------------------

Alternative within S1 is Type-II blow-up, which allows a non-standard blow-up rate, for example:

  u(x, t) ~ (T - t)^(-α) V( x / (T - t)^β , τ )

with exponents α, β not equal to 1/2.

This leads to modified rescaled equations with additional time-dependent terms. Potential phenomena:
  - Ancient solutions that do not reduce to steady states in τ.
  - Heteroclinic orbits between equilibria in the rescaled dynamics.

This direction may sidestep some self-similarity non-existence theorems but at the cost of more complex PDE structure.

10. Final Branch Summary
------------------------

Summary of our choice tree:

  1. Global regularity vs blow-up:
     - Chose blow-up construction (B).

  2. Blow-up strategies:
     - B1: Self-similar/DSS (general).
     - B2: Axisymmetric with swirl (chosen).
     - B3: Convex integration.

  3. Within B2:

  4. Within B2.1:

     - Chose S1.

  5. Within S1:

     - S1-T2: Type-II blow-up (non-standard rates).
     - We leaned toward S1-DSS as a concrete analytic target, with S1-T2 as an additional possibility.

This log captures:
  - The equations at each stage,

  - The idea of recasting potential blow-up as a periodic-orbit problem in the rescaled Navier–Stokes flow on a suitable function space.

SECTION 10: PHYSICAL INTERPRETATION - FIXED POINTS AND REPEATING PATTERNS
================================================================================

Q: "How does fluid utilize a fixed point?"

A: A fixed point of the Poincaré map P(v*) = v* means that in RESCALED VARIABLES,
   the velocity field returns to itself after one period T_λ = 2log(λ):
   
   v*(y) = v(y, T_λ)
   
   In ORIGINAL VARIABLES, this corresponds to DISCRETE SELF-SIMILARITY:
   
   u(x,t) = λ u(λx, T + λ²(t-T))
   
   PHYSICAL MEANING:
   - At times t = T - λ^(-2k), the flow pattern repeats at scales λ^k (smaller)
     and intensities λ^(-k) (stronger)
   - The fluid doesn't "stay still" - it evolves, but in rescaled coordinates,
     it follows a CLOSED LOOP (limit cycle) in function space
   - The fixed point v* is the TEMPLATE PATTERN that repeats at each discrete
     time step
   
   EXAMPLE with λ=2:
   - t = T - 1: pattern at scale 1
   - t = T - 1/4: same pattern at scale 1/2 (twice as small, twice as fast)
   - t = T - 1/16: same pattern at scale 1/4 (four times smaller, four times faster)
   - As t → T, the pattern shrinks and intensifies, repeating the same shape
   
   CONNECTION TO "CIRCLES" THEME:
   - In similarity variables, the fluid state moves on a CLOSED LOOP in the
     infinite-dimensional space of all possible flows
   - This is a "circle in function space" - the fixed point is the point on
     that loop that repeats after one period

SECTION 11: CASCADE BLOW-UP STRUCTURE - HIERARCHICAL SINGULARITIES
================================================================================

CONCEPTUAL CORRECTION:
  Original assumption: Simple repeating self-similar pattern
  Corrected structure: A HIERARCHICAL CASCADE of interacting singularities

STRUCTURE:
  1. PRIMARY SINGULARITY:
     - Decreasing "action weight" (energy/intensity)
     - Increasing "area of effect" (spatial extent)
     - Located at (x₁, T₁) with T₁ = T
  
  2. SECONDARY SINGULARITIES:
     - Interact off the primary
     - Converging/separating wave functions
     - Reduced weight and area of effect
     - Located at (x_k, T_k) with T_k ≥ T
  
  3. WAVE-MEDIATED INTERACTION:
     - Pressure waves: Δp = -∇·(u·∇u)
     - Vorticity waves: ∂_t ω + (u·∇)ω = (ω·∇)u + νΔω
     - These waves propagate between singular points and trigger secondaries

CASCADE BLOW-UP ANSATZ:
  u(x,t) = u₁(x,t) + Σ_{k=2}^N u_k(x,t) + u_int(x,t)
  
  where:
  - u₁ = primary singularity (weakening, spreading)
  - u_k (k≥2) = secondary singularities (weaker, smaller)
  - u_int = wave-mediated interaction field

PRIMARY SINGULARITY STRUCTURE:
  For primary at (x₁, T₁) with T₁ = T:
  
  u₁(x,t) = (T-t)^(-α₁) U₁((x-x₁)/(T-t)^(β₁))
  
  Constraints:
  - Decreasing intensity: α₁ ≤ 0 (or very small positive)
  - Increasing scale: β₁ > 0
  - Energy constraint: α₁ + (3/2)β₁ ≤ 0 (roughly, for finite energy)
  
  THIS IS NON-STANDARD: Typical blow-up has α₁ > 0, β₁ > 0
  
  Possible interpretations:
  - α₁ = 0: Constant intensity, spreading scale
  - α₁ < 0: Weakening intensity, spreading scale
  - Energy might be conserved or transferred to secondaries

SECONDARY SINGULARITIES:
  For secondary k at (x_k, T_k) with T_k ≥ T:
  
  u_k(x,t) = (T_k - t)^(-α_k) U_k((x-x_k)/(T_k - t)^(β_k)) · χ_k(t)
  
  where:
  - α_k < α₁ (weaker)
  - β_k < β₁ (smaller scale) or possibly β_k > β₁ (spreading faster)
  - χ_k(t) = activation function (0 before trigger, 1 after)
  
  TRIGGER MECHANISM: Secondary k activates when a wave from the primary
  (or another secondary) reaches a threshold

WAVE-MEDIATED INTERACTION:
  The interaction field u_int satisfies a wave equation driven by singularities.
  
  From NSE, pressure satisfies:
  Δp = -∇·(u·∇u) = -Σ_{i,j} ∂_i u_j ∂_j u_i
  
  Near singularities, this becomes:
  Δp = -Σ_{k=1}^N ∇·(u_k·∇u_k) - cross terms - interaction terms
  
  The pressure waves propagate and can trigger secondaries.
  
  VORTICITY WAVES: The vorticity equation
  ∂_t ω + (u·∇)ω = (ω·∇)u + νΔω
  also propagates vorticity from primary to secondaries.
  
  We model u_int as solving:
  ∂_t u_int + ∇p_int = νΔu_int + F_wave
  where F_wave is the wave forcing from the singularities

CONSISTENCY EQUATIONS:
  Plugging cascade ansatz into NSE:
  ∂_t u + (u·∇)u = -∇p + νΔu,  ∇·u = 0
  
  gives:
  
  1. For each u_k: A profile equation (similar to self-similar, but cascade scaling):
     -α_k U_k - β_k y_k·∇U_k + (U_k·∇)U_k = -∇P_k + ΔU_k
     where y_k = (x-x_k)/(T_k-t)^(β_k)
  
  2. For u_int: A forced wave equation with sources from all u_k
  
  3. Cross-coupling terms: Interactions between different u_k via pressure and advection

CONSTRUCTION METHOD OPTIONS:
  
  OPTION C1: ITERATIVE CONSTRUCTION (BOTTOM-UP)
  - Start with primary u₁ (solve its profile equation)
  - Compute the wave field it generates
  - Identify where waves exceed threshold → place secondary u₂
  - Repeat: compute waves from u₁ + u₂ → place u₃, etc.
  - Show the iteration converges to a finite cascade
  
  OPTION C2: FIXED-POINT ON FULL CASCADE
  - Define a map F that takes candidate cascade {u_k} and:
    * Computes the wave field
    * Updates positions/intensities of secondaries based on wave thresholds
    * Solves profile equations for each u_k
  - Find a fixed point of F (the cascade is self-consistent)
  
  OPTION C3: VARIATIONAL / ENERGY CASCADE
  - Treat the cascade as an energy transfer process
  - Primary loses energy → waves → secondaries gain energy
  - Formulate as optimization or constrained minimization
  
  STATUS: Choice pending (C1, C2, or C3)

SECTION 15: FIXED-POINT CONSTRUCTION METHOD (C2) - FORMAL SETUP
================================================================================

CHOICE: Fixed-Point on Full Cascade (C2)

RATIONALE:
  - Most mathematically rigorous (can use Schauder/Brouwer fixed-point theorems)
  - Handles all interactions simultaneously (not sequential like C1)
  - Natural for proving existence of self-consistent cascade

FUNCTION SPACES FOR CASCADE COMPONENTS:

PRIMARY SINGULARITY SPACE X₁:
  For primary singularity u₁, work in similarity variables:
  v₁(y,τ) = (T-t)^(1/2) u₁(x,t),  y = (x-x₁)/√(T-t),  τ = -log(T-t)
  
  Define:
  X₁ = { v₁ ∈ H¹_ax(ℝ³) | ∇·v₁ = 0,  ||v₁||_{X₁} < ∞ }
  
  Norm includes:
  - Standard L² and H¹ terms
  - Weighted decay: |v₁(y)| ≲ |y|^(-1-δ) as |y|→∞ (for finite energy)
  - Regularity near r=0: axisymmetric fields with controlled behavior at axis
  
  KEY CONSTRAINT: For finite-energy initial data:
  ∫ |u₁(x,0)|² dx = ∫ |v₁(y,τ₀)|² e^(-3τ₀/2) dy < ∞
  This imposes decay conditions on v₁

SECONDARY SINGULARITY SPACE X_k (k ≥ 2):
  For secondary k at (x_k, T_k), define rescaled variables:
  v_k(y_k,τ_k) = (T_k-t)^(α_k + 1/2) u_k(x,t)
  y_k = (x-x_k)/(T_k-t)^(β_k),  τ_k = -log(T_k-t)
  
  Define:
  X_k = { v_k ∈ H¹_ax(ℝ³) | ∇·v_k = 0,  ||v_k||_{X_k} < ∞,  supp(v_k) localized }
  
  Constraints:
  - α_k < α₁ (weaker intensity)
  - β_k chosen to ensure finite energy and proper scaling
  - Localization: v_k decays rapidly outside region of size ~ (T_k-t)^(β_k)

INTERACTION FIELD SPACE X_int:
  The wave-mediated interaction field:
  X_int = { v_int ∈ L²_loc(ℝ³ × [0,T)) | ∇·v_int = 0,  wave-like decay }
  
  Captures:
  - Pressure waves: solutions of Δp = -∇·(u·∇u)
  - Vorticity waves: solutions of vorticity transport equation
  - Far-field behavior: waves propagate with finite speed

THE FIXED-POINT MAP F:
  Define F: X → X where X = X₁ × ∏_{k=2}^N X_k × X_int:
  
  F({v_k}, v_int) = ({v_k*}, v_int*)
  
  where:
  
  1. FOR EACH v_k: Solve profile equation in rescaled variables:
     -α_k v_k - β_k y_k·∇v_k + (v_k·∇)v_k = -∇P_k + Δv_k + forcing from waves
     subject to ∇·v_k = 0
  
  2. COMPUTE WAVE FIELD: Given {v_k}, solve:
     Δp_wave = -Σ_{k=1}^N ∇·(v_k·∇v_k) - cross terms
     and
     ∂_t v_int + ∇p_int = νΔv_int + F_wave[{v_k}]
     where F_wave is wave forcing computed from all v_k
  
  3. UPDATE SECONDARY POSITIONS/INTENSITIES: Based on wave thresholds:
     - If |p_wave(x_k)| > threshold_k, activate secondary k
     - Adjust α_k, β_k, x_k, T_k to balance wave forcing and energy constraints
  
  4. RETURN UPDATED CASCADE: ({v_k*}, v_int*)

FIXED-POINT EQUATION:
  A self-consistent cascade is a fixed point:
  F({v_k*}, v_int*) = ({v_k*}, v_int*)
  
  This means:
  - Each singularity profile v_k* is consistent with wave field it generates
  - Wave field is consistent with all singularities
  - Secondary positions/intensities are self-consistently determined

EXISTENCE VIA SCHAUDER FIXED-POINT THEOREM:
  To prove existence, need:
  
  1. COMPACTNESS: Show F maps bounded, convex, closed set K ⊂ X into itself,
     and F(K) is relatively compact
  
  2. CONTINUITY: Show F is continuous on K
  
  3. INVARIANT SET: Construct K such that F(K) ⊂ K
  
  KEY CHALLENGES:
  - Wave equation is non-local (pressure determined by elliptic equation)
  - Cross-coupling between singularities creates nonlinear dependencies
  - Energy constraints must be maintained throughout

CASCADE PROFILE EQUATIONS (DETAILED):

  For primary singularity v₁ in rescaled variables (y, τ):
  ∂_τ v₁ = Δv₁ - (1/2)y·∇v₁ - (1/2)v₁ - (v₁·∇)v₁ - ∇π₁ + F₁[v_int]
  where F₁[v_int] is forcing from interaction field

  For secondary k in its own rescaled variables (y_k, τ_k):
  ∂_{τ_k} v_k = Δv_k - α_k v_k - β_k y_k·∇v_k - (v_k·∇)v_k - ∇π_k
                + F_k[{v_j}_{j≠k}, v_int]
  where F_k includes:
  - Wave forcing from primary and other secondaries
  - Cross-advection terms
  - Activation function χ_k(τ_k)

WAVE THRESHOLD MECHANISM:
  Define wave intensity function:
  I_wave(x,t) = |∇p_wave(x,t)|² + |ω_wave(x,t)|²
  where:
  - p_wave is pressure field from all singularities
  - ω_wave is vorticity field
  
  Secondary k activates at position x_k and time T_k when:
  I_wave(x_k, T_k) > Θ_k
  where Θ_k is threshold (typically decreasing: Θ_k < Θ_{k-1})
  
  Activation function:
  χ_k(t) = { 0  if t < T_k or I_wave(x_k, t) < Θ_k
           { 1  if t ≥ T_k and I_wave(x_k, t) ≥ Θ_k

ENERGY CASCADE CONSTRAINTS:
  To maintain finite total energy:
  Σ_{k=1}^N E_k(t) + E_int(t) ≤ E_0
  where:
  - E_k(t) = ∫ |u_k(x,t)|² dx is energy of singularity k
  - E_int(t) = ∫ |u_int(x,t)|² dx is interaction energy
  - E_0 is initial total energy
  
  For primary with α₁ ≤ 0, β₁ > 0:
  E₁(t) = (T-t)^(-2α₁ - 3β₁) ∫ |U₁(y)|² dy
  
  For finite energy: -2α₁ - 3β₁ ≤ 0, i.e., α₁ + (3/2)β₁ ≤ 0
  
  For secondaries:
  E_k(t) = (T_k-t)^(-2α_k - 3β_k) ∫ |U_k(y_k)|² dy_k · χ_k(t)
  
  Cascade structure suggests:
  - Primary energy decreases: E₁(t) → 0 as t → T
  - Secondary energies increase but remain bounded: E_k(t) ≤ C_k E_0
  - Total energy is conserved or decreases (viscous dissipation)

RIGOROUS JUSTIFICATION OF WEIGHTED SPACES:

  Weighted L² space:
    L²_w = { v : ∫ |v(y)|² (1+|y|²)^δ dy < ∞ },  δ > 3/2
